Tezos Mobile SDKs – Demo

Swift (iOS) and Kotlin (Android) SDKs with Beacon (TZIP-10) integration.

Quick Actions

iOS (Swift Package)

.package(url: "https://github.com/oxdev6/tezos-mobile-sdk.git", from: "0.1.0")

Open GitHub Repository

Android (Gradle)

implementation("io.tezos:mobile-sdk:0.1.0")

Open GitHub Repository

Fetch Head Hash (Swift)

let rpc = TezosRPCClient(config: .init(rpcURL: URL(string: "https://mainnet.tezos.marigold.dev")!))
let head = try await rpc.getHeadHash()
print(head)

Fetch Head Hash (Kotlin)

val rpc = TezosRpcClient("https://mainnet.tezos.marigold.dev")
val head = rpc.getHeadHash()
println(head)
